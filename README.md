# Trasmilenio_copy

Repositorio destinado a la gestión y despliegue de recursos relacionados con Trasmilenio.

## Estructura del Proyecto

```
/
├── .Gitignore            # Configuración de archivos ignorados por Git
├── .env                  # Variables de entorno (configuración sensible)
├── README.md             # Este archivo, documentación principal del proyecto
├── scripts/              # Scripts auxiliares (automatizaciones, despliegue, etc.)
└── src/                  # Código fuente principal de la aplicación
```

## Tecnologías utilizadas

- **CSS**: 52.8%
- **HTML**: 31.9%
- **Shell (Bash/Scripts)**: 15.3%

## Descripción de Carpetas y Archivos

- `.Gitignore`: Lista archivos y carpetas que no serán versionados.
- `.env`: Variables de entorno, típicamente para configuración sensible o local.
- `README.md`: Documentación general del proyecto.
- `scripts/`: Scripts utilizados para tareas administrativas, automatización o despliegue.
- `src/`: Aquí se aloja el código fuente principal del proyecto.

## Cómo empezar

1. Clona el repositorio:
   ```bash
   git clone https://github.com/Juan-Pablo-Castillo-Velasquez/Trasmilenio_copy.git
   ```
2. Configura el archivo `.env` según sea necesario.
3. Explora el contenido en las carpetas `src` y `scripts` para entender la lógica y los automatismos.
